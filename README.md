vim-pytype
----------

A simple vim plugin for executing [Pytype](https://github.com/google/pytype)



Install
-------
[vim-plug](https://github.com/junegunn/vim-plug)
```
Plug 'XiaoConstantine/vim-pytype'
```


Commands
--------
|Command   |List                                      |
|---       |---                                       |
|:Pytype   | Run pytype check on the current buffer   |
